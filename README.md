# web-design-lab questiobns

## Program list:

*1. Write html code to develop a web page having two frames that divide the webpage into two equal
rows and then divide the row into equal columns and fill each frame with a different background color*

*2. Design web pages for your college containing a description of the courses, departments, faculties, library etc, use href, list tags.*

*3.Create your resume using HTML tags and also experiment with colors, text, link, size and also other tags you studied.*

*4.Design a web page of your home town with an attractive background color, text color, an Image, font, etc. (use internal CSS).*

*5. Design a blog layout using HTML and CSS*

*6.Write a JavaScript program to calculate and display the area of a rectangle. Prompt the user to enter the length and width of the rectangle, and then compute the area using the formula: area =length * width.*
